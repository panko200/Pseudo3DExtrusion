疑似3D押し出し  
製作者：Panko200  
配布場所：https://github.com/panko200/Pseudo3DExtrusion

## 概要

奥行き方向に押し出しできます。

## 使用方法

映像エフェクト「疑似3D押し出し」が追加されます。  
この映像エフェクトよりも以前に、カメラエフェクトや3D回転エフェクトを置き、  
疑似3D押し出しを適用すると、カメラの値を動かすと立体になってます。死ぬほど重いです。  
内部でシェイプ化してるので、ちょっと形が変わることがあります。  
カメラや3D回転のパラメータの値によっては、表示が崩れる場合があります。

## アンインストール方法

1. YMM4を起動して`ヘルプ(H)`>`その他`>`プラグインフォルダを開く`をクリックする。
2. YMM4を終了する。
3. `Pseudo3DExtrusion`という名前のフォルダを削除する。

## 注意点

動作確認済みのバージョンは、ゆっくりムービーメーカーv4.50.0.2です。

他のSkiaSharpを使うプラグインと、使用するバージョンが違う場合、プラグインが競合する可能性があります。試してないけど。
